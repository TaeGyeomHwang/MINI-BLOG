### mini-blog
 # 개요

리액트를 활용해서 미니 블로그를 제작했다.

해당 미니 프로젝트는 list, page, ui 컴포넌트로 구성되며, 주요 기능은 다음과 같다.

- 글 목록 보기 기능(리스트 형태)
- 글 보기 기능
- 댓글 보기 기능
- 글 작성 기능
- 댓글 작성 기능
